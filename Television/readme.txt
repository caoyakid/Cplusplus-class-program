Purpose:
Create a class called Television that has the member variables:
displayType – a string that store display  types (e.g., LCD, OLED, QLED, ..)
dimension  --  a double variable for storing dimension of the TV in inches
connectivitySupport – a dynamic array of strings that stores the different connectivity modes supported by the TV. 
(e.g., Component Video, Composite Video, Digital Video Interface (DVI), FireWire (IEEE-1394 connection, iLink), 
High-Definition Multimedia Interface (HDMI), S-Video (mini-DIN), RJ-45 Ethernet Cable,..)
In the end, ask the user to if customization is required for any television.
If so, use the mutator methods to set the values accordingly.
